jQuery Events

Attach and Detach Event Handlers

To attach :

$(‘class/id/tag’).on(“” , function(){});

To detach :

.off(event [,selectors] [,handler] )
$(this).off() — detach all the events attached to calling object
$(this).off(‘click’) — detach the click event but keeps other event
